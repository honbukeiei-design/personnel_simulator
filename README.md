# 病床機能・経営データ統合 人員要望意思決定支援アプリ

添付の経営・人員パネル、`01_（別紙1）申請表`、厚生労働省の病床機能報告オープンデータを統合して、人員要望の承認判断を支援する Streamlit アプリです。

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 同梱データ

- `data/financial_hospital_panel.xlsx`: 添付の経営・人員パネル
- `data/application_form.xlsx`: 添付の申請表テンプレート/申請書

画面左のアップローダーから差し替え可能です。

## 主な機能

1. 申請表の主要項目・コメント欄のUI表示
2. 病院別の最新年度経営・人員データ抽出
3. 職種別の対病床人員、ピア中央値、不足人数の算定
4. 増員人数・人件費・増収感応度・政策価値重みのシナリオ分析
5. MHLW令和6年度病床機能報告オープンデータのリンク取得・試読込
6. フィルタ済みデータのCSV出力

## 注意

- MHLWサイトへの接続は実行環境のネットワークに依存します。接続できない場合は、病床機能報告ページからExcelを手動でダウンロードし、アプリ側へアップロードしてください。
- 増収感応度や政策・質安全価値の重みは仮説パラメータです。意思決定会議で係数を調整してください。
